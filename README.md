# Project-Magang-Peta-Laju-Perkembangan-Desa-dan-Kelurahan
Peta Laju Perkembangan Desa dan Kelurahan Berdasarkan Hasil Evaluasi Pembangunan Kabupaten Purwakarta Tahun 2024

## Deskripsi
Repository ini hasil kegiatan magang yang dilakukan di Dinas Komunikasi dan Informatika Kabupaten Purwakarta. Peta laju perkembangan desa dan kelurahan disusun untuk menampilkan
perubahan dan perkembangan tingkat pembangunan wilayah desa dan kelurahan berdasarkan hasil evaluasi pembangunan. Informasi yang di sajikan pada peta ialah batas administrasi wilayah dan klasifikasi laju perkembangan. Peta IDM digunakan sebagai media visualisasi untuk menggambarkan kondisi pembangunan desa secara umum.

## Kegiatan
- Pengolahan dan penyajian data spasial
- Penyusunan peta tematik

## Tools
- QGIS
- ArcGIS
- Microsoft Excel

Peta Laju Perkembangan Desa dan Kelurahan Berdasarkan Hasil Evaluasi Pembangunan Kabupaten Purwakarta Tahun 2024
